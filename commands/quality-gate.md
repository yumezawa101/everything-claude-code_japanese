# Quality Gate コマンド

ファイルまたはプロジェクトスコープに対して ECC 品質パイプラインをオンデマンド実行します。

## 使用方法

`/quality-gate [path|.] [--fix] [--strict]`

- デフォルト対象: カレントディレクトリ (`.`)
- `--fix`: 設定されている場合、自動フォーマット/修正を許可
- `--strict`: サポートされている場合、警告でも失敗とする

## パイプライン

1. 対象の言語/ツールを検出
2. フォーマッタチェックを実行
3. lint/型チェックが利用可能な場合は実行
4. 簡潔な修正リストを生成

## 注意事項

このコマンドは hook の動作をミラーしますが、ユーザーが手動で呼び出します。

## 引数

$ARGUMENTS:
- `[path|.]` オプションの対象パス
- `--fix` オプション
- `--strict` オプション
