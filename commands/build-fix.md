# Build and Fix

TypeScriptおよびビルドエラーを段階的に修正：

1. ビルドを実行: npm run build または pnpm build

2. エラー出力を解析：
   - ファイルごとにグループ化
   - 重大度でソート

3. 各エラーについて：
   - エラーのcontext（前後5行）を表示
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決したことを確認

4. 以下の場合は停止：
   - 修正が新しいエラーを導入した
   - 3回試行しても同じエラーが続く
   - ユーザーが一時停止を要求

5. サマリーを表示：
   - 修正されたエラー
   - 残っているエラー
   - 新たに導入されたエラー

安全のため、一度に1つのエラーを修正すること！
