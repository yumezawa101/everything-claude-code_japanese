# リファレンス帰属とライセンスに関する注記

ECC v1.8.0 は以下のリサーチとワークフローのインスピレーションを参照しています:

- `plankton`
- `ralphinho`
- `infinite-agentic-loop`
- `continuous-claude`
- 公開プロフィール: [zarazhangrui](https://github.com/zarazhangrui), [humanplane](https://github.com/humanplane)

## ポリシー

1. ライセンスされていない、または互換性のないソースからの直接コードコピーは禁止。
2. ECC の実装は本リポジトリのアーキテクチャとライセンスモデルに合わせて再作成されている。
3. 参照された素材は、ライセンスが明示的に再利用を許可しない限り、アイデア、パターン、概念的フレーミングのみに使用。
4. 将来の直接再利用には、明示的なライセンス検証とファイル内およびリリースノート内のソース帰属が必要。
