# ECC v1.8.0 リリースノート

## ポジショニング

ECC v1.8.0 は、プロジェクトを単なる設定バンドルではなく、agent ハーネスパフォーマンスシステムとして位置付けます。

## 主な改善点

- Hook とライフサイクル動作の安定化。
- 評価とループオペレーションサーフェスの拡張。
- 運用使用に向けた NanoClaw のアップグレード。
- クロスハーネスパリティの改善（Claude Code、Cursor、OpenCode、Codex）。

## アップグレードの焦点

1. 環境で hook プロファイルのデフォルトを検証。
2. `/harness-audit` を実行してプロジェクトのベースラインを設定。
3. `/quality-gate` と更新された eval ワークフローを使用して一貫性を強制。
4. 参照エコシステムの帰属とライセンスノートを確認: [reference-attribution.md](./reference-attribution.md)。
5. パートナー/スポンサー向けの視覚表現には、ライブ配布メトリクスとトーキングポイントを使用: [../business/metrics-and-sponsorship.md](../../business/metrics-and-sponsorship.md)。
